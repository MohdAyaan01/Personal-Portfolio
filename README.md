## ✨ Overview
A premium, high-performance personal portfolio built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. This portfolio showcases my journey as a Full-Stack Developer, featuring sophisticated motion graphics, glassmorphic design, and a curated selection of AI-powered projects.

---

## 🛠️ Tech Stack

### Frontend
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Animations:** CSS Keyframes & Framer Motio

## 🌟 Key Features

- **Dynamic Hero Section:** Featuring massive ambient glows and high-impact typography.
- **Starry Background:** A custom SVG-animated celestial backdrop.
- **Project Showcases:** Detailed cards for major projects like **Neuro Glow AI** and **Pro Folio X**.
- **Interactive Experience:** Smooth transitions, hover-state interactions, and glassmorphic UI components.
- **Responsive Design:** Optimized for all screen sizes, from mobile to ultra-wide monitors.

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MohdAyaan01/Portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
